Teach ADAM — Continual Learning With Approval Gates

Type a short phrase. ADAM computes a gradient that would reduce its loss on that phrase, but does not apply it — the update is staged and shown to you. You decide whether ADAM actually learns from it. No other production AI does this.

Hebbian Memory — Engrams Stored Inside the Weights

Type a distinctive phrase. ADAM embeds it and writes it to its Hebbian memory M (a buffer inside the model's state_dict — NOT an external database). The top-5 slots show cosine similarity to what you just wrote. If you write a similar phrase later, the same slot lights up.

# ADAM — A Dynamical Alive Mind

Interactive demo of ADAM, a 52M-param transformer with features no production LLM has:

- **Hebbian memory inside model weights** (no external DB)
- **Constraint identity attractor I\*** — a mathematical "self"
- **Live 5-axis emotion** projected from the state vector every forward pass
- **Approval-gated continual learning** — you decide when it learns
- **Theory-of-mind second state vector** tracking the user
- **World model** with forward dynamics, reward, termination, value heads
- **PAT consciousness benchmark** — ADAM scores H-CDB Class 4

## Four playable demos

1. **Teach ADAM** — propose a weight update, approve or reject
2. **Identity Attractor** — 2D PCA visualization of S(t) orbiting I\*
3. **Memory Probe** — write phrases, watch them land in M slots
4. **H-CDB Arcade** — five consciousness tests
